# LoginDemo: 누구나 따라 할 수 있는 만들기 설명서 (초등학생도 가능!)

> **이 설명서는 여러분이 "컴퓨터 레고 조립"을 하듯이 처음부터 끝까지 따라만 하면 멋진 앱과 서버를 만들 수 있도록 아주 자세하게 적혀 있어요.**
> 어려운 말이 나와도 걱정 마세요. 그대로 따라 치기만 하면 됩니다!

---

## 📅 목차 (순서대로 따라오세요)

1. **준비물 챙기기** (내 컴퓨터에 필요한 도구 설치)
2. **1단계: 프로젝트 가져오기** (시작 버튼 누르기)
3. **2단계: 설계도 만들기** (무엇을 만들지 정하기)
4. **3단계: 서버 만들기** (데이터 저장소 짓기)
5. **4단계: 아이폰 앱 만들기** (iOS)
6. **5단계: 안드로이드 앱 만들기** (Android)
7. **6단계: 인터넷에 공개하기** (친구들에게 자랑하기)

---

## 🎒 1. 준비물 챙기기

요리를 하려면 칼과 도마가 필요하듯이, 개발을 하려면 도구가 필요해요. 아래 프로그램들이 없으면 먼저 설치해주세요.

### 필수 프로그램 설치하기
1. **VS Code** (코드를 적는 공책): [다운로드 링크](https://code.visualstudio.com/)
2. **Node.js** (자바스크립트 실행기): [다운로드 링크](https://nodejs.org/) (LTS 버전으로 받으세요!)
3. **Docker** (서버 컨테이너): [다운로드 링크](https://www.docker.com/products/docker-desktop/)
4. **Xcode** (아이폰 앱 공장): 맥북의 App Store에서 'Xcode' 검색해서 설치.
5. **Android Studio** (안드로이드 앱 공장): [다운로드 링크](https://developer.android.com/studio)

## 🚀 2단계: 프로젝트 가져오기

이제 진짜 시작해볼까요? 터미널(검은 화면)을 열고 아래 명령어를 한 줄씩 복사해서 붙여넣으세요.

### 1. 내 컴퓨터에 방 만들기
```bash
# 내 문서 폴더로 이동하기
cd Documents

# 'LoginDemo'라는 이름의 새 폴더(방) 만들기
mkdir LoginDemo

# 그 방으로 들어가기
cd LoginDemo
```

### 2. 소스 코드 가져오기 (복사해오기)
이미 만들어진 기본 틀을 가져올 거예요.
```bash
# 깃허브에서 코드 복사해오기
git clone https://github.com/dyjung/logindemo.git .

# 잘 가져왔는지 확인하기 (목록 보기)
ls -al
```
*`typespec`, `ios`, `android`, `backend` 같은 폴더 이름들이 보이면 성공이에요!*

## 📝 3단계: 설계도 만들기 (기획)

우리가 무엇을 만들지 컴퓨터에게 미리 알려주는 단계예요. 우리는 **"명세 주도 개발"**이라는 멋진 방법을 쓸 거예요. 이건 **"설계도 먼저, 건물을 나중에"** 짓는 규칙이에요.

### 1. 헌법(규칙) 확인하기
`speckit`이라는 도구가 우리를 도와줄 거예요. 먼저 우리가 지킬 규칙을 확인해봐요.
```bash
# .specify 폴더 안에 규칙이 잘 있는지 눈으로 확인만 하세요.
cat .specify/memory/constitution.md
```

### 2. 어떤 기능을 만들지 정하기 (스펙)
로그인 기능을 만든다고 가정해볼게요.
1. `specs/001-login` 폴더를 만들어요.
2. 그 안에 `spec.md` 파일을 만들고 "이메일 로그인 기능을 만들고 싶어"라고 적어요.
(하지만 지금은 이미 만들어진 파일들이 있으니 넘어갈게요!)

## ⚙️ 4단계: 서버 만들기 (백엔드)

앱이 데이터를 저장하고 불러올 수 있는 '본부'를 만드는 과정이에요.

### 1. 백엔드 폴더로 이동
```bash
cd typespec/backend
```

### 2. 필요한 부품 조립하기 (설치)
```bash
# 필요한 모든 부품을 다운로드 받아요
npm install
```
*화면에 막 글자가 지나가고 `added ... packages`라고 뜨면 된 거예요.*

### 3. 데이터베이스(창고) 연결하기
서버가 데이터를 저장할 비밀 창고를 설정해야 해요.
```bash
# 예제 파일을 복사해서 진짜 설정 파일 만들기
cp .env.example .env

# 데이터베이스 창고 문 열기
npx prisma db push

# 서버가 창고를 쓸 수 있게 도구 만들기
npx prisma generate
```

### 4. 서버 켜기!
```bash
npm run start:dev
```
*초록색 글씨로 `Nest application successfully started`가 떴나요? 축하합니다! 서버가 켜졌어요.*
*(다음 단계를 위해 터미널 새 창을 열어주세요)*

## 📱 5단계: 아이폰 앱 만들기 (iOS)

이제 화면에 보이는 앱을 만들어봐요. 맥북이 있어야 가능해요!

### 1. Xcode 열기
터미널에서 아래 명령어를 치거나, 파인더에서 파일을 더블클릭하세요.
```bash
cd ../../ios
open LoginDemo.xcodeproj
```

### 2. 시뮬레이터 실행하기
1. Xcode가 열리면 상단 가운데에 `LoginDemo > iPhone 15` 처럼 적힌 부분이 있어요. 그걸 눌러서 원하는 아이폰 기종을 고르세요.
2. 키보드에서 **`Command` + `R`** 키를 동시에 누르세요. (재생 버튼 ▶️을 눌러도 돼요)
3. 잠시 기다리면 화면에 가짜 아이폰이 뜨고 앱이 실행될 거예요!

## 🤖 6단계: 안드로이드 앱 만들기

안드로이드 폰용 앱도 만들어봐요.

### 1. Android Studio 열기
Android Studio를 실행하고 **Open** 버튼을 누른 뒤, 우리가 받은 `LoginDemo` 폴더 안에 있는 `android` 폴더를 선택하세요.

### 2. 기다리기 (중요!)
처음 열면 오른쪽 아래에 막대기가 움직이면서 뭔가를 계속 다운로길 거예요. 이게 다 끝날 때까지 얌전히 기다려야 해요. (커피 한 잔 하고 오세요 ☕️)

### 3. 실행하기
상단에 초록색 재생 버튼(▶️)을 누르면 가짜 안드로이드 폰(에뮬레이터)이 뜨면서 앱이 켜져요.

## 🌍 7단계: 인터넷에 공개하기 (배포)

내 컴퓨터에서만 되는 게 아니라, 친구들도 쓸 수 있게 진짜 인터넷 서버에 올리는 방법이에요. **Docker**라는 "컨테이너 박스" 기술을 쓸 거예요.

### 1. 터미널에서 프로젝트 제일 바깥으로 나가기
```bash
cd ../..
# 지금 위치가 LoginDemo 폴더인지 확인하세요
pwd
```

### 2. 마법의 주문(Docker) 외우기
이 명령어 하나면 서버, 데이터베이스, 웹사이트가 한 번에 다 켜져요.
```bash
# Docker야, 조립해서 실행해줘! (백그라운드 모드)
docker compose up -d
```

### 3. 잘 됐는지 확인하기
```bash
docker compose ps
```
*`Up` 이라고 적혀있으면 성공! 이제 인터넷 주소창에 `http://localhost:3000`을 쳐보세요.*

## 🚑 문제 해결 (안 될 때 보세요)

**Q. 명령어를 쳤는데 'command not found'라고 나와요!**
A. 1단계 '준비물 챙기기'에서 해당 프로그램을 설치하지 않아서 그래요. 설치하고 터미널을 껐다 다시 켜보세요.

**Q. 권한(Permission)이 없다고 나와요!**
A. 명령어 맨 앞에 `sudo`를 붙여보세요. (예: `sudo docker compose up ...`) 비밀번호를 물어보면 맥북 로그인 비밀번호를 입력하면 돼요.
